# Arduino TTC Shield

## Overview
This Arduino shield was made for testing and learning TTC systems. It combines both the nRF24L01 and LoRa RA-02 modules on a single board, making it easy to switch between short- and long-range wireless communication during development.

The board also includes an INA219 for power tracking, understanding the relation between Link-Budget and Power-Budget.

## Features
- Arduino Uno shield form factor
- nRF24L01 wireless module interface
- LoRa RA-02 module interface
- INA219 current sensing
- Jumper-selectable power for nRF24L01
- Jumper-selectable power for LoRa
- Boost converter for battery operation
- Battery connector
- Power indicator LED

## Applications
- TTC (Telemetry, Tracking and Command) development
- learning LoRa RA-02 and NRF24L01 
- Power consumption measurement of RF modules
- Rapid prototyping with Arduino
